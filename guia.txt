Este arquivo eh um guia basico de como rodar o projeto:

- Na pasta API-MERCADINHO so irá precisar abrir o terminal e inserir : 'npm run dev' para funcionar .

- Na pasta mercado-front vc precisará intalar o node_modules para funcionar corretamente 'npm install' ou 'npm i' e em seguida o comando 'npm run dev'

importante: Certifique-se de estar no diretório correto antes de rodar os comandos.(pois API-MERCADINHO ja tem node_modules)
